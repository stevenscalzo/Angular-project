¿Quiénes somos?

En la actualidad, al momento de querer realizar compras por internet, contamos con una gran variedad de tiendas y opciones. Muchas pueden llegar a ofrecer un mismo producto o uno muy similar por un precio muy distinto, por lo que se hace complicado poder comparar los productos de distintas webs. Es por ello por lo que esta web ofrece al cliente la oportunidad de ver los productos de distintas tiendas online en una sola web, pudiendo realizar la comparación y elección de forma sencilla.

Este proyecto tiene como objetivo general crear una e-commerce donde el producto ofrecido será ropa deportiva proveniente de e-commerces reales. Permitirá realizar la búsqueda por categorías, nombre de producto, filtrado por precios y por web que ofrece el producto. Se permitirá al usuario realizar la compra en esta web o poder acceder al producto original en la tienda de origen. Contará con registro de usuarios, acceso (login), carrito de la compra, perfil de usuario con el historial de compra. Para ello, nos trazamos los siguientes puntos como objetivos:

Objetivos principales:

  • Distribución de productos por categorías.
  • Registro de nuevos usuarios, acceso (login) mediante correo y contraseña.
  • Filtrado de productos por palabras en nombre o descripción, por precios y/o por web que ofrece el producto.
  • Permitir añadir productos al carrito y realizar la compra.
  • Desarrollar un servidor que ofrezca mediante API la información de productos y usuarios.

Objetivos secundarios:

  • Permitir editar datos de usuarios.
  • Permitir eliminar/editar contenido en el carrito de compras.
  • Validaciones de formularios.
  • Diseño y experiencia de usuario adecuada.

El trabajo realizado es el Proyecto de Fin de Ciclo del módulo de Desarrollo de Aplicaciones Web.

La finalidad del proyecto es profundizar y emplear el conocimiento sobre el framework Angular y sobre el desarrollo de servicios Node y Express. Se intentará realizar un proyecto que pueda ser empleado para una empresa en la vida real.

La herramienta para el desarrollo del proyecto será Visual Studio Code donde se utilizarán los lenguajes Typescript, HTML y SCSS para Angular y el lenguaje Javascript para el desarrollo en Node. El desarrollo se realizará con los conocimientos adquiridos en el módulo DAW utilizando las clases vistas como principal material de apoyo.
